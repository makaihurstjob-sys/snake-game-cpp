# Snake Game (C++)

## Project Overview
This project is a simple Snake game written in C++. The goal is to practice object-oriented programming concepts such as classes, encapsulation, and modular design while building a functional game.

## Team Roles
- Game Logic: Snake movement, collisions
- Game Board: Grid and boundaries
- Food System: Food spawning and scoring
- Main Driver: Game loop and input handling
